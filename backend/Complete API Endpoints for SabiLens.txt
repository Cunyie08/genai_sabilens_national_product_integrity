Complete API Endpoints for SabiLens

1. Authentication Endpoints
Public Auth
POST /api/auth/register		- Register new user (consumer/company/nafdac) POST /api/auth/login	- Login with email/phone & password
POST /api/auth/logout	- Logout user
POST /api/auth/refresh-token	- Refresh access token POST /api/auth/forgot-password		- Request password reset
POST /api/auth/reset-password		- Reset password with token POST /api/auth/verify-email	- Verify email address
POST /api/auth/verify-phone		- Verify phone with OTP POST /api/auth/resend-otp	- Resend verification OTP

Response Formats
Success Response
```json
{
"success": true, "data": {},
"message": "Operation successful", "timestamp": "2024-01-15T14:30:00Z"
}

Error Response
```json
{
"success": false, "error": {
"code": "ERROR_CODE",
"message": "Error description", "details": {}
},
"timestamp": "2024-01-15T14:30:00Z"
}
```


Paginated Response
```json
{
"success": true, "data": [], "pagination": { "page": 1,
"limit": 20,
"total": 100,
"pages": 5
}
}
